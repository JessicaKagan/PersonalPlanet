# Personal Planet

Heads up - this project is in pre-alpha! It'll become more complete and elaborate over time. The goal for _Personal Planet_ remains what it was in my initial planning/brainstorming document - an incremental game loosely inspired by Maxis' old "software toys", and SimEarth in particular.

As of June 18th, 2026, this project is online more to document a personal project I'm working on as time and energy permit. I'm building from a Phaser/Svelte template project; see `TEMPLATE_README.md` for the current setup instructions. You can expect online hosted builds in the future, when I've had more time and energy to design and build out this toy.
